1.1. Открытие файла для записи
Открыть файл с именем filename в режиме записи:
ofstream file(filename)

Если файл не удалось открыть → вывести сообщение об ошибке и завершить функцию:
if (!file) → cerr << "Ошибка открытия файла: " << filename

1.2. Генерация и запись 10 случайных чисел
Для i от 0 до 9 (включительно, 10 итераций):
for (int i = 0; i < 10; ++i)

Сгенерировать случайное целое число от 1 до 10 по формуле:
randomNumber = rand() % 10 + 1
rand() % 10 даёт число от 0 до 9 → +1 сдвигает диапазон на 1 → [1, 10] 

Записать randomNumber в файл:
file << randomNumber

Если i < 9 (то есть не последнее число), добавить пробел после числа:
if (i < 9) → file << " "

1.3. Закрытие файла
Закрыть файл после записи:
file.close()

Шаг 2: Основная программа (main)

Вызвать функцию generateRandomNumbers трижды для создания трёх разных файлов:
generateRandomNumbers("file1.txt")
generateRandomNumbers("file2.txt")
generateRandomNumbers("file3.txt")

Вывести сообщение об успешном завершении:
cout << "Файлы успешно созданы!" << endl

Завершить программу с кодом возврата 0 (успех):
return 0
